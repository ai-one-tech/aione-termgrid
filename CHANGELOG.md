# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-05-23

### Added

- 🎉 **初始版本发布**
  - 网格布局支持（1x1 到 4x4）
  - 单元格合并功能
  - 主题跟随（自动切换深色/浅色模式）
  - 多语言界面（中/英双语支持）
  - 拖动分割线调整大小
  - 终端搜索功能
  - 单元格放大模式
  - 批量启动/停止/重启终端
  - 平台特定命令支持（Windows/macOS/Linux）
  - 延迟启动功能
  - 自定义边框颜色
  - 另存为功能
  - 侧边栏集成
  - 命令面板集成

### Features

- **Terminal Management**
  - 网格布局中的多个终端
  - 独立控制每个终端
  - 批量操作所有终端
  - 终端状态显示（运行中/已停止/等待中/错误）

- **UI/UX**
  - 响应式网格布局
  - 单元格合并预览
  - 拖动调整大小
  - 最大化单个终端
  - 搜索功能（大小写敏感、全词匹配）
  - 状态指示器（脉冲动画）

- **Configuration**
  - YAML 配置文件格式
  - 平台特定命令
  - 启动顺序控制
  - 延迟启动
  - 自定义边框颜色
  - 工作目录设置

- **Integration**
  - VS Code 侧边栏
  - 命令面板
  - 自定义编辑器
  - 主题跟随
